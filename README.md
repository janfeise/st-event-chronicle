# 📜 Event Chronicle — SillyTavern 扩展

从 AI 角色扮演对话中自动提取结构化事件，构建可视编年史时间线，注入长期记忆。

## 安装

在 SillyTavern 扩展管理中，粘贴以下 URL：

```
https://github.com/janfeise/Event-Chronicle
```

无需额外配置，克隆即用。

## 功能

- **自动事件提取** — 每 N 条消息自动调用 LLM 提取事件，无需手动操作
- **长期记忆注入** — 每次 AI 生成回复前注入编年史，避免遗忘早期情节
- **自动合并去重** — 累计 M 条新事件自动整理，保持编年史精简
- **时间线浏览** — 可视化查看所有事件，按地点分组，支持搜索和筛选
- **事件编辑** — 支持修改/删除事件，数据即时持久化
- **一键批量生成** — 从聊天历史增量生成事件，切片处理避免超上下文

## 配置

在 SillyTavern 扩展设置面板中调整：

| 配置项     | 默认值    | 说明                                       |
| ---------- | --------- | ------------------------------------------ |
| 提取阈值   | 10 条消息 | 累计消息数达到后自动触发事件提取           |
| 整理阈值   | 5 个事件  | 累计新事件数达到后自动合并去重             |
| 最大 Token | 2048      | 覆盖 ST 默认 maxToken，确保 JSON 输出完整  |
| 模型设置   | 继承 ST   | 可选独立配置事件提取模型（建议用便宜模型） |
| 批量切片   | 5 条/批   | 一键生成时每批发送的消息数                 |

## 注意事项

- 事件提取和合并会消耗 API Token
- 批量生成历史数据前会弹出 Token 消耗提醒
- 修改模型设置后需刷新页面使配置生效
